# Architecture Documentation

This directory is the **system of record** for architectural decisions.
If agents can't see it here, it doesn't exist to them.

## Contents
- Domain maps and package layering
- Dependency direction rules
- Integration patterns

## Principle (Harness Engineering H2)
> "Slack discussions, Google Docs, tribal knowledge — none of it exists to an agent.
>  Everything must be encoded into the repository as versioned, discoverable artifacts."
